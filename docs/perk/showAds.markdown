## Overview

Shows a Perk ad based on a placement&nbsp;ID from the [Perk Developer](http://developers.perk.com) dashboard.


## Syntax

	perk.showAds( placementID )

##### placementID ~^(required)^~
_[String][api.type.String]._ An ad placement&nbsp;ID.
